To generate  unit tests for the container class using Randoop, you need to do the following steps:
1- Open Command Prompt (for windows)
2- Go to the C:\
3- Create workspace folder 
4- Copy Randoop folder to c:\workspace folder
5- Compile the container class (i.e., mvn compile)
6- Go to the directory container\randooptests
7- run the following command 
      java -classpath .;C:\WorkSpace\Randoop\container\target\classes\;c:\WorkSpace\Randoop\randoop-all-3.0.8.jar randoop.main.Main gentests --classlist=c:\WorkSpace\Randoop\container\randooptests\myclasses.txt --timelimit=10 
8- Copy the java files, which are generated by Randoop, from the randooptests and paste them to the src\test\java\container
9- Run mvn test and mvn jacoco:report

Note:
If you decide to change the paths or the folder names, change the command-line accordingly.
